# 🚀 Own Cursor IDE

> Build stunning websites using AI — just type a prompt and watch magic happen ✨

---

## 🧠 What is this?

**Own Cursor IDE** is an AI-powered website builder that generates complete websites (HTML, CSS, JS) from a simple prompt.

💡 Just tell what you want → AI builds it instantly.

---

## ⚡ Features

* 🤖 AI Website Generation (HTML, CSS, JS)
* 🎨 Modern UI (Dark theme + responsive)
* ⚡ Instant Live Preview
* 🛠 Tool-based file creation system
* ⚡ Ultra fast runtime using **Bun**

---

## 🧪 How it Works

1. Enter a prompt
2. AI generates code
3. Files are created automatically
4. Website loads instantly in preview

---

## 💬 Example Prompts

```bash
Create a modern portfolio website
```

```bash
Build a calculator website with buttons and animations
```

```bash
Make a landing page for a startup with navbar and hero section
```

👉 💥 **Tip:**
The better your prompt → the better your website

---

## 🚀 How to Run (Using Bun)

### Install dependencies

```bash
bun install
```

### Start server

```bash
bun index.js
```

Open in browser:

```bash
http://localhost:3001
```
